### 2. Write a C program and calculate following additions. Argue what happens in each one and why do you see the particular result
#### a. INT_MIN + INT_MIN
Prints out:
```
INT_MIN + INT_MIN = 0
```

What happens is that it takes the two smallest signed integers ($-2^{7}$ or 1000 0000) and adds them together which creates an overflow, where the carrybit will be deleted and the number resets to zero.

#### b. INT_MIN + INT_MIN
Prints out:
```
INT_MIN + INT_MAX = -1
```

Adding the minimum (1000 0000) number to the maximum number (0111 1111) fills out all the bits with 1, which results in -1.

#### c. INT_MIN + INT_MIN
Prints out:
```
INT_MAX + INT_MAX = -2
```

Adding the largest integer twice (0111 1111) creates a ripple of carry's, which fills in all bits with 1 except the first slot (1111 1110) which is equal to -2.
